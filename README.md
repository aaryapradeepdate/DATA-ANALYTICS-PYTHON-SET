# 📊 Retail Price Data Analytics Project

## 📌 Project Overview
This project focuses on **analyzing OLTP (Online Transaction Processing) dataset** to uncover meaningful insights related to price distribution, variability, and patterns across products. The goal is to demonstrate practical **data analytics skills** using Python, including data cleaning, exploratory data analysis (EDA), visualization, and insight generation.

---

## 🎯 Objectives
* Analyze retail price data to understand pricing trends
* Identify variations, outliers, and patterns in product prices
* Apply data analytics techniques to a real-world retail dataset
* Generate actionable insights relevant to retail and market analysis

---

## 🗂️ Dataset Description
* **Domain:** Retail / Market Analytics
* **Type:** Structured tabular data
* **Key Attributes:**
- TransactionID, Date, Time, CustomerID, Name, Surname, Shipping_State, Item No. , Description, Retail_Price and      
Loyalty_Discount  

> The dataset was preprocessed and cleaned to ensure accuracy and reliability before analysis.

---

## 🛠️ Tools & Technologies Used
* **Programming Language:** Python
* **Environment:** Jupyter Notebook
* **Libraries used:**
  * pandas – data manipulation
  * numpy – numerical operations
  * matplotlib – data visualization
  * seaborn – advanced visual analytics

---

## 🔄 Project Workflow

1. **Data Import & Understanding**

   * Loaded the dataset (.csv) file and examined structure, data types, and basic statistics

2. **Data Cleaning & Preprocessing**

   * Checked for missing values and duplicates
   * Standardized column formats

3. **Exploratory Data Analysis (EDA)**
   * Performed basic and descriptive numerical/statistical calculations
   * Performed top 10 analysis as per the following -
      1) Top 10 Customers by Spending
      2) Top 10 Products by Revenue
      3) Top 10 States by Sales
      4) Time Based Analysis (based on hour)
      5) Top 10 Highest Daily Sales
      6) Top 10 Loyalty Discount Dates
   * Price distribution analysis
   * Identification of outliers for fraud and anomaly detection

4. **Data Visualization**
   * Presented the data through visuals using data visualization techniques to show the previous listed findings in a better way. 
   * Comparative visuals for better interpretation.

5. **Insights & Interpretation**
   * Extracted business-relevant insights from pricing trends

---

## 📈 Key Insights
* Retail prices show noticeable variation, indicating differences in product sales.
* Presence of outliers suggests sales as per premium or products purchased by using loyalty discount.
* Price distribution patterns can help in understanding consumer affordability, state-level sales distribution and market segmentation
* Repeat customers drive significant revenue, highlighting loyalty program effectiveness.
* Product concentration: a few products dominate revenue.
* Seasonal peaks in transaction volume suggest promotional opportunities.
